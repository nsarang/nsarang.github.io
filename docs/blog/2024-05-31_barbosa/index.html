Multi-Headed Attention (MHA)

This is a tutorial/implementation of multi-headed attention from paper Attention Is All You Need in PyTorch. The implementation is inspired from Annotated Transformer.

Here is the training code that uses a basic transformer with MHA for NLP auto-regression.

Here is an experiment implementation that trains a simple transformer.

Prepare for multi-head attention

This module does a linear transformation and splits the vector into given number of heads for multi-head attention. This is used to transform key, query, and value vectors.

8class PrepareForMultiHeadAttention(nn.Module):
# 
 9    def __init__(self, d_model: int, heads: int, d_k: int, bias: bool):
10        super().__init__()
#

Linear layer for linear transform

11        self.linear = nn.Linear(d_model, heads * d_k, bias=bias)
#

Number of heads

12        self.heads = heads
#

Number of dimensions in vectors in each head

13        self.d_k = d_k
# 
14    def forward(self, x: torch.Tensor):
#

Input has shape [seq_len, batch_size, d_model] or [batch_size, d_model]. We apply the linear transformation to the last dimension and split that into the heads.

15        head_shape = x.shape[:-1]
#

Linear transform

16        x = self.linear(x)
#

Split last dimension into heads

17        x = x.view(*head_shape, self.heads, self.d_k)
#

Output has shape [seq_len, batch_size, heads, d_k] or [batch_size, heads, d_model]

18        return x
#

Multi-Head Attention Module

This computes scaled multi-headed attention for given query, key and value vectors.

$$\mathop{Attention}(Q, K, V) = \underset{seq}{\mathop{softmax}}\Bigg(\frac{Q K^\top}{\sqrt{d_k}}\Bigg)V$$

In simple terms, it finds keys that matches the query, and gets the values of those keys.

It uses dot-product of query and key as the indicator of how matching they are. Before taking the \(softmax\) the dot-products are scaled by \(\frac{1}{\sqrt{d_k}}\). This is done to avoid large dot-product values causing softmax to give very small gradients when \(d_k\) is large.

Softmax is calculated along the axis of of the sequence (or time).

19class MultiHeadAttention(nn.Module):
#
  • heads is the number of heads.
  • d_model is the number of features in the query, key and value vectors.
20    def __init__(
21        self,
22        heads: int,
23        d_model: int,
24        dropout_prob: float = 0.1,
25        bias: bool = True,
26    ):
27
28        super().__init__()
#

Number of features per head

29        self.d_k = d_model // heads
#

Number of heads

30        self.heads = heads
#

These transform the query, key and value vectors for multi-headed attention.

31        self.query = PrepareForMultiHeadAttention(
32            d_model, heads, self.d_k, bias=bias
33        )
34        self.key = PrepareForMultiHeadAttention(
35            d_model, heads, self.d_k, bias=bias
36        )
37        self.value = PrepareForMultiHeadAttention(
38            d_model, heads, self.d_k, bias=True
39        )
#

Softmax for attention along the time dimension of key

40        self.softmax = nn.Softmax(dim=1)
#

Output layer

41        self.output = nn.Linear(d_model, d_model)
#

Dropout

42        self.dropout = nn.Dropout(dropout_prob)
#

Scaling factor before the softmax

43        self.scale = 1 / math.sqrt(self.d_k)
#

We store attentions so that it can be used for logging, or other computations if needed

44        self.attn = None
#

Calculate scores between queries and keys

This method can be overridden for other variations like relative attention.

45    def get_scores(self, query: torch.Tensor, key: torch.Tensor):
#

Calculate \(Q K^\top\) or \(S_{ijbh} = \sum_d Q_{ibhd} K_{jbhd}\)

46        return torch.einsum("ibhd,jbhd->ijbh", query, key)
#

mask has shape [seq_len_q, seq_len_k, batch_size], where first dimension is the query dimension. If the query dimension is equal to \(1\) it will be broadcasted.

47    def prepare_mask(
48        self,
49        mask: torch.Tensor,
50        query_shape: List[int],
51        key_shape: List[int],
52    ):
53
54        assert mask.shape[0] == 1 or mask.shape[0] == query_shape[0]
55        assert mask.shape[1] == key_shape[0]
56        assert mask.shape[2] == 1 or mask.shape[2] == query_shape[1]
#

Same mask applied to all heads.

57        mask = mask.unsqueeze(-1)
#

resulting mask has shape [seq_len_q, seq_len_k, batch_size, heads]

58        return mask
#

query, key and value are the tensors that store collection of query, key and value vectors. They have shape [seq_len, batch_size, d_model].

mask has shape [seq_len, seq_len, batch_size] and mask[i, j, b] indicates whether for batch b, query at position i has access to key-value at position j.

59    def forward(
60        self,
61        *,
62        query: torch.Tensor,
63        key: torch.Tensor,
64        value: torch.Tensor,
65        mask: Optional[torch.Tensor] = None
66    ):
#

query, key and value have shape [seq_len, batch_size, d_model]

67        seq_len, batch_size, _ = query.shape
68
69        if mask is not None:
70            mask = self.prepare_mask(mask, query.shape, key.shape)
#

Prepare query, key and value for attention computation. These will then have shape [seq_len, batch_size, heads, d_k].

71        query = self.query(query)
72        key = self.key(key)
73        value = self.value(value)
#

Compute attention scores \(Q K^\top\). This gives a tensor of shape [seq_len, seq_len, batch_size, heads].

74        scores = self.get_scores(query, key)
#

Scale scores \(\frac{Q K^\top}{\sqrt{d_k}}\)

75        scores *= self.scale
#

Apply mask

76        if mask is not None:
77            scores = scores.masked_fill(mask == 0, float("-inf"))
#

\(softmax\) attention along the key sequence dimension \(\underset{seq}{softmax}\Bigg(\frac{Q K^\top}{\sqrt{d_k}}\Bigg)\)

78        attn = self.softmax(scores)
#

Save attentions if debugging

79        tracker.debug("attn", attn)
#

Apply dropout

80        attn = self.dropout(attn)
#

Multiply by values $$\underset{seq}{softmax}\Bigg(\frac{Q K^\top}{\sqrt{d_k}}\Bigg)V$$

81        x = torch.einsum("ijbh,jbhd->ibhd", attn, value)
#

Save attentions for any other calculations

82        self.attn = attn.detach()
#

Concatenate multiple heads

83        x = x.reshape(seq_len, batch_size, -1)
#

Output layer

84        return self.output(x)
